Автоматичне резервне копіювання

Windows:
1. Відкрити Task Scheduler.
2. Обрати Create Basic Task.
3. Вказати назву, наприклад Database Backup.
4. Обрати періодичність запуску (щодня або щотижня).
5. У дії обрати Start a Program.
6. У полі Program/script вказати:
powershell.exe
7. У полі Add arguments вказати:
-ExecutionPolicy Bypass -File "C:\Project\scripts\backup-db.ps1"
8. Завершити створення завдання.
Після цього резервна копія буде створюватися автоматично відповідно до обраного розкладу.

Linux:
1. Відкрити термінал.
2. Виконати команду:
crontab -e
3. Додати рядок:
0 2 * * * cd /path/to/project && ./scripts/backup-db.sh
У наведеному прикладі резервна копія створюватиметься щодня о 02:00.
4. Зберегти зміни та закрити редактор.
Після цього система автоматично виконуватиме створення резервних копій згідно з налаштованим розкладом.
